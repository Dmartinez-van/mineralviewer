# Mineral Viewer

SPA for viewing different minerals. Uses ThreeJS to create the scene and load models. Models were downloaded forom [EDUROCK_AALTO](https://sketchfab.com/EDUROCK_AALTO) via their [SketchFab](https://sketchfab.com/) pages

Site should be available to vist at -> [whatarock.site](www.whatarock.site)

## Purpose

[Whatarock.site](www.whatarock.site) was made as a simple education introduction to different minerals. Information about their chemical componds, their structures, where they are found, and more are planned to be added to the viewer.

Hope this can help someone learn a bit more about mineralogogy.
